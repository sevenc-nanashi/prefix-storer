# Prefix Viewer

このBotはBotのプレフィックスを保存し、一括でニックネームを変更できるBotです。

## 使い方

`/register` でBotのプレフィックスを登録します。
`/show` で登録したプレフィックスを表示します。
`/pnick format` でニックネームのフォーマットを変更します。
`/pnick change` でBotのニックネームを変更します。

### フォーマット

| フォーマット | 例 |
| ----------- | --- |
| `[ %p ] %n` | `[ / ] Prefix Viewer` |
| `<%p> %n` | `</> Prefix Viewer` |
| `%p ) %n` | `/ ) Prefix Viewer` |
| `%p> %n` | `/ > Prefix Viewer` |
